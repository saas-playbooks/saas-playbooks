## Open-source tools & resources

Free, MIT-licensed companions to the playbooks:

| Repo | What it is |
|------|------------|
| 📚 [**awesome-saas-engineering**](https://github.com/saas-playbooks/awesome-saas-engineering) | A curated list of resources for building and running a SaaS, organized by topic — each entry links to a full playbook. |
| ✅ [**saas-launch-checklist**](https://github.com/saas-playbooks/saas-launch-checklist) | The complete launch checklist, from architecture to production. Every item links to a step-by-step playbook. |
| 💻 [**saas-playbooks-cli**](https://github.com/saas-playbooks/saas-playbooks-cli) | Search and open the playbooks from your terminal — a zero-dependency Node.js CLI. |
| 🔴 [**saas-incident-runbooks**](https://github.com/saas-playbooks/saas-incident-runbooks) | Ready-to-use incident-response runbooks for SaaS outages — 502s, failed payments, DB errors, and more. |
